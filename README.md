"# Chamador de Senhas - Sistema de Convocações 

Sistema web para chamada de senhas/convocações da Prefeitura Municipal de Juiz de Fora.

## 🚀 Novidades - Sistema de Administração

### ✨ Interface Visual para Upload de Arquivos
Agora você pode gerenciar os arquivos Excel de forma visual, sem precisar de conhecimento técnico!

**Acesse:** `admin.html` ou clique no botão "⚙️ ADMIN" na navbar

### 📋 Funcionalidades do Admin

- **📁 Upload de Arquivos**: Arraste e solte ou clique para selecionar arquivos Excel
- **🔗 Associação Automática**: Vincule arquivos às páginas de forma visual
- **👁️ Prévia de Dados**: Visualize o conteúdo dos arquivos antes de usar
- **✅ Teste do Sistema**: Verifique se tudo está funcionando corretamente
- **💾 Backup de Configurações**: Salve e restaure suas configurações

## 📖 Como Usar

### 1. Configuração Inicial (Nova Forma - Fácil!)
1. Abra `admin.html` no navegador
2. Faça upload dos seus arquivos Excel (.xlsx)
3. Associe cada arquivo à página correspondente
4. Teste o sistema para verificar se tudo está correto

### 2. Configuração Tradicional (Antiga - Para Programadores)
- Coloque os arquivos Excel na pasta raiz do projeto
- Os nomes devem seguir o padrão já estabelecido no código

## 🎮 Controles Durante o Uso

- **→ (Seta Direita)**: Próxima pessoa
- **← (Seta Esquerda)**: Pessoa anterior  
- **P**: Tocar áudio de chamada
- **1-9**: Definir número do guichê

## 🆘 Solução de Problemas

### Arquivo não encontrado?
1. Acesse `admin.html`
2. Verifique se o arquivo foi carregado corretamente
3. Confirme se está associado à página certa
4. Use a função "Testar Sistema"

### Dados não aparecem?
1. Verifique se o Excel tem a estrutura correta (Ordem | Nome)
2. Certifique-se que não há linhas vazias no início
3. Use a prévia no admin para verificar os dados

### Sistema lento?
- Arquivos muito grandes podem demorar para carregar
- Considere dividir listas muito extensas
